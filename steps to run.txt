Steps to run with docker compose - local
1-./infrastructure/scripts/setup-env.sh
2-cp ./.env.dev infrastructure/docker/.env #depends enviroment
3-docker compose -f infrastructure/docker/docker-compose.yml up -d

Steps to run within docker container - local
1-./infrastructure/scripts/setup-env.sh
2-npm install
3-
docker compose -f infrastructure/docker/docker-compose.yml up -d # to create mariadb container
or
configure .env for connection to mariadb and initialize database
4-npm run dev

Steps to run with kubernetes - stage
1-./infrastructure/scripts/setup-env.sh
2-./infrastructure/k8s/scripts/init-staging.sh
3-./infrastructure/scripts/build-and-push-docker-image.sh infrastructure/docker/Dockerfile

Steps to run terraform
1-cd infrastructure/terraform
2-cp terraform.tfvars.example terraform.tfvars
3-Change terraform.tfvars to your needs
4-./scripts/get-aws-resources.sh # check your AWS resources
5-terraform init
6-terraform plan
7-terraform apply
